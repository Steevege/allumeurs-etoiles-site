# Guide - Gérer le site Allumeurs d'Étoiles

Tout se gère depuis le CMS (interface admin du site) : les dates de tournée, les actualités, les articles et les spectacles.

**Adresse du CMS** : https://allumeurs-detoiles.netlify.app/admin/

---

## Se connecter au CMS

1. Va sur **https://allumeurs-detoiles.netlify.app/admin/**
2. Clique sur **"Sign In with GitHub"**
3. Connecte-toi avec ton compte GitHub

---

## 1. Gérer les dates de tournée

### Ajouter une date

1. Dans le menu de gauche, clique sur **Dates de tournée**
2. Clique sur **"New"** en haut à droite
3. Remplis les champs :

| Champ | Obligatoire ? | Quoi mettre | Exemple |
|-------|:---:|-------------|---------|
| **Spectacle** | Oui | Nom du spectacle ou événement | Festychant |
| **Date** | Oui | Clique sur le calendrier pour choisir | 15/03/2026 |
| **Ville** | Oui | Ville avec département | Angers (49) |
| **Lieu** | Non | Nom de la salle (optionnel) | Théâtre Municipal |
| **Heure** | Non | Heure de début (optionnel) | 20h30 |

4. Clique **"Save"** en haut à droite
5. Le site se met à jour automatiquement en ~2 minutes

### Modifier une date

1. Clique sur **Dates de tournée** dans le menu
2. Clique sur la date à modifier
3. Modifie les champs souhaités
4. Clique **"Save"**

### Supprimer une date

1. Clique sur la date à supprimer
2. Clique **"Delete"** en haut à droite
3. Confirme la suppression

### Les dates se rangent automatiquement

- Les dates futures apparaissent dans **"Prochaines dates"**
- Les dates passées se déplacent dans **"Dates passées"**
- Les 3 prochaines dates s'affichent sur la **page d'accueil**
- Tu n'as rien à faire, c'est automatique

---

## 2. Publier une actualité

1. Dans le menu de gauche, clique sur **Actualités**
2. Clique sur **"New"** en haut à droite
3. Remplis :
   - **Titre** : le titre de l'actu
   - **Date publication** : date du jour
   - **Résumé court** : 1-2 phrases max (s'affiche sur la page d'accueil)
   - **Afficher sur homepage ?** : coche Oui si tu veux que ça apparaisse sur la page d'accueil
   - **Image** : clique pour uploader une photo (optionnel)
   - **Catégorie** : choisis entre Actualités, Coulisses ou Témoignages
   - **Contenu complet** : écris ton article
4. Clique **"Save"**
5. Le site se met à jour en ~2 minutes

---

## 3. Publier un article de blog

1. Clique sur **Articles Blog** dans le menu
2. Clique sur **"New"**
3. Remplis :
   - **Titre** : titre de l'article
   - **Date publication** : date du jour
   - **Auteur** : ton nom ou "Allumeurs d'Étoiles"
   - **Résumé** : 2-3 phrases qui donnent envie de lire
   - **Image mise en avant** : photo pour illustrer (optionnel)
   - **Catégorie** : Actualités, Coulisses ou Témoignages
   - **Contenu** : écris ton article
4. Clique **"Save"**

---

## 4. Utiliser l'éditeur de texte

L'éditeur fonctionne comme un traitement de texte simple :

- **Gras** : sélectionne le texte et clique sur **B**
- **Italique** : sélectionne le texte et clique sur **I**
- **Lien** : sélectionne le texte, clique sur l'icône lien, colle l'URL
- **Image** : clique sur l'icône image pour en ajouter une dans le texte
- **Liste** : clique sur l'icône liste à puces

### Ajouter une image

1. Clique sur le champ image
2. Choisis **"Upload"** pour envoyer une photo depuis ton ordinateur
3. Formats acceptés : JPG, PNG
4. Conseil : images pas trop lourdes (max 1-2 Mo)

## En cas de problème

- **Le site ne se met pas à jour** : attends 2-3 minutes, le déploiement est automatique
- **Tu ne peux pas te connecter au CMS** : vérifie ton compte GitHub
- **Un champ est en rouge** : il est obligatoire, remplis-le avant de sauvegarder
- **Autre problème** : contacte Steeve
